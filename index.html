CSS Colors

What is this?

After years of coloring, both with and without my kids, and writing website code, I decided it would be cool to be able to use the same colors as my trusty box of 64 crayons. The only problems?
  • The built-in usable "web colors" are limited to the standard naming conventions.
  • There are a large variety of colors in the Crayola boxes, that have custom names.
  • While sites like W3, and random others have created color lists, they weren't really usable (or reusable), as you would need to enter the Hex values every time.
After some quick web searching, i found that W3Schools had done the basic framework, by collecting all the colors in one place. The only problem was that you had to code the colors manually using the hex code (e.g. #84DE02). As I had been using the W3 CSS framework for a while now, I thought, wouldn't it be cool if I could do the same.
